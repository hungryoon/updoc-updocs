# updoc — Wiki

<!-- updoc:begin -->
updoc is a documentation management tool that keeps your project docs up to date automatically. When code changes, updoc detects the differences and regenerates the relevant documentation sections — so your docs never go stale.

It works as a Claude Code plugin: you run a simple command (`/updoc:up`), and it scans your codebase to produce two kinds of documentation:
- **Technical docs** for developers (architecture, configuration, dependencies)
- **Wiki docs** for everyone else (features, access, policies)

updoc also helps with planning. The `/updoc:uplan` command reads your current documentation and helps you create structured mission documents for new features or changes, with clear task breakdowns.
